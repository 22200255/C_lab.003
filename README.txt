레포트에서는 차마 적을 수 없었던 TMI 및 궁금했던 점 등 적어보겠습니다. 
만약 이런걸 적은게 불편하거나 그래서는 안된다면 즉각 삭제처리 하겠습니다... 

사실 코딩 레포트를 처음 써보기도 해서 인터넷에 검색해서 그나마 비슷한 걸 따라써봤습니다.. 
혹시 레포트 형식을 알려주신다면 다음 과제부터 즉각 반영하여 성심성의껏 만들어보겠습니다.
모자란 C언어 실력때문에 알고리즘을 짜도 구현을 못해 시간을 많이 들였습니다. 
챗지피티한테 물어보면 답이 바로 나오지만 자존심이 상해 알려준 대로가 아닌 최대한 제가 생각한 방식대로 구현하게 해봤습니다. 
코드도 복붙이 아닌 필요한 함수만 가져다 쓰는 식으로 했습니다!
while(1)의 무한루프를 까먹어서(...) 재귀함수를 썼습니다.
막상 이렇게 얼렁뚱땅하다보니 코딩이란게 투박하면 안되지만 저만의 투박한 코드가 만들어졌다는게 뭔가 보람찹니다.
코딩에 걸린 시간보다 github 연동하는게 더 오래걸리고 힘들었던 것 같습니다... 무서운건 영상봐도 이해가 안돼서 뭔가 많이 만졌는데 어찌저찌 된 거입니다..
